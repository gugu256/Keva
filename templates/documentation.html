Keva

Keva is a simple key-value database inspired by replit's DB.

You can use Keva by sending simple HTTP GET requests to the server. Here are the endpoints :

Base URL : keva.pancakedev.repl.co. Replace the db_key argument by your database's key provided earlier.


/new/db_key?key=username&value=gugu256 Adds a new key-value pair to your database, replace the query arguments by the key and value of your key-value pair
Every value is assumed to be a string at first, but you can add the &type= at the end of your query to specifiy the type. The supported types are int, float, str, list, dict, bool and json
Example :
/new/db_key?key=age&value=29&type=int


/upd/db_key?key=age&value=30&type=int Updates a key in your database, it works the exact same way as /new and takes the same arguments


/get/db_key?key=age Returns a JSON object containing the key you provided and its value. If you do not provide any key, this endpoint will return your whole database as a JSON object


/getkeys/db_key Returns a JSON list containing all the keys of your database


/del/db_key?key=age Deletes the key your provided from your database

/reset/db_key Resets your database to an empty dictionary. Note : this does not delete your database, it just clears it, if you want to delete your database, go to the homepage and click "Delete my database"
