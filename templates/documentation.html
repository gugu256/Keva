Keva

Keva is a simple key-value database inspired by replit's DB.

You can use Keva by sending simple HTTP GET requests to the server. Here are the endpoints :

Base URL : keva.pancakedev.repl.co. Replace the db_key argument by your database's key provided earlier.


/set/db_key/key/value/type Adds/Updates a key-value pair to your database, replace the query arguments by the key and value of your key-value pair
You have to add the type of the data you're sending so the server understands it. The supported types are int, float, str, list, dict, bool and json
Example :
/new/7w7a6SOkukH996Ax/username/gugu256/str


/get/db_key/key Returns a JSON object containing the key you provided and its value


/get_keys/db_key Returns a JSON list containing all the keys of your database


/get_entire/db_key Returns a JSON object containing your whole database


/del/db_key/key Deletes the key your provided from your database

/reset/db_key Resets your database to an empty dictionary. Note : this does not delete your database, it just clears it, if you want to delete your database, go to the homepage and click "Delete my database"
